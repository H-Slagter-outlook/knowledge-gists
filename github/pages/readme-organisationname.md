# README.md

Profile information for an organisation.<br/>
Displayed when opening https://github.com/organisation.

Stored in __.github__ repo which needs to be a public repo.

Path: __profile/README.md__

## Members only

For users that have been added as members to the organisation there is an option to display a members only profile README.

This __README.md__ file needs to be stored in the __profile__ folder of a private repo named __.github-private__.
