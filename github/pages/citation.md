# CITATION.cff

This file gives guidance on how to cite the work that is stored in the repo, or the authors of the work, etc.
It can also be used to cite work used in creating the work stored in the repo.

The format is yaml based.

it needs to be stored in the root folder of the main branch of the repo.

Example can be found at citation-file-format.github.io

Supported formats:
- APA
- BibTeX
