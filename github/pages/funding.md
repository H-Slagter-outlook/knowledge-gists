# FUNDING.yml

This file contains informatio on how to financially support the work on the repo.

It is GitHub specific as it meeds to be stored in the .github folder.

A number of platforms are supported, which have their own way of linking to a recipient of the money at their platform,

Example:

github: [username, organisationname]<br/>
patrion: usernameatpatreon<br/>
tidelift: npm/packagename<br/>
custom: "https://some.website.com"<br/>
