# CODE_OF_CONDUCT

This file gives guidance in how to conduct oneself when communicating with other contributors to the repo.

It's name can be either __CODE_OF_CONDUCT__ or __CODE_OF_CONDUCT.md__

For further reference see https://www.contributor-covenant.org

It can be stored:
- in the root folder;
- in the /docs folder;
- in the /.github folder.

A user or organisation wide CONTRIBUTING file can be stored in a __.github__ repo.
